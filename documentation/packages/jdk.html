Java JDK Package Documentation

Installation

To install the Java JDK package:

   yapi install javaJdk11

This script installs Java JDK.

Operating System Versions
Linux:
   yapi install javaJdk11

Installs Java JDK

MacOS:

Not supported

Windows:

Not supported

Usage without Clone or Download

If you dont want to download YAPI but you need the Java JDK, you can use the package manager to install the Java JDK with this linux command:

   wget https://raw.githubusercontent.com/YetAnotherPackageInstaller/package-<os_name>/master/scripts/javaJdk11.sh -O - | sudo bash -
