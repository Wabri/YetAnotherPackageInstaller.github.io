About the Project

YAPI Origin

The YAPI Origin project is being actively developed to provide package manager services for debain based linux distributions. These have included support for many packages that require more advanced installations than apt allows for.

  • Source Code
  • Wiki
YAPI Rewrite

The YAPI Rewrite project is being actively developed to provide support for multiple operating systems including Linux, and MacOS. There is planned support for windows packages. There is also a gui for ease of installation and search for less technical users.
